# 🛸 Рик и Морти — Межпространственная База Данных

Приложение на React, собранное с помощью Vite и React Router v7. Содержит персонажей, локации и эпизоды из вселенной Рика и Морти.

## 🚀 Быстрый старт

### Требования

- Node.js 18+
- npm 9+

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/Alex-Kukharenko/rickandmorty.git

# Перейти в папку проекта
cd rickandmorty

# Установить зависимости
npm install

# Запустить сервер разработки
npm run dev
```

## ⚙️ Функциональность

- **React Router v7** — вложенные маршруты, динамические сегменты, `useParams`, `useNavigate`
- **Сортировка через query-параметры** — сортировка по `created` ASC/DESC, отражается в URL (`?sort=asc`)
- **Общий макет** — навигация остаётся на всех страницах через `<Outlet />`
- **Страница 404** — перехватывает все неизвестные маршруты
- **Локальные JSON данные** — без внешних API запросов
